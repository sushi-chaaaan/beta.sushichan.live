## はじめに

先日、VLL の一大ライブイベントである 「MIKUEC 2024」 が無事に開催されました。
オフライン会場では 3 公演開催して合計 1000 人以上の方にご来場いただき、最終公演のみだった YouTube 配信でも最大同時視聴者数 1000 人を超えるなど大盛況のライブとなりました。
あらためて、ご来場 / ご視聴いただいた皆様に感謝申し上げます。

<UrlCard url="https://mikuec.com/2024/" />

このライブをもって、私を含む 2022 年度入学生は第一線を退くことになります。
私が班長を務めていた技術班でも代替わりを行う時期となり、ほとんどを私が担当していた仕事も徐々に後輩に任せる割合を増やしていく必要があります。

これは、自分で触る部分が多いからこそ成立していた「属人化しているが当人の中では最も効率良い」仕事の形を変えていく必要があることを意味します。
なぜなら、一人でまずまずの成果を出すためのやり方と、複数人のチームで協働しより大きな成果を出していくためのやり方は大きく異なるからです。

### ところで技術班とは

実は技術班は 2024 年度からできた新しい班です。

もともと VLL の Web サイトやチャットツール内の Bot、部内 Wiki などは有志の部員によって維持されていました。
しかし、運用の難しさ、人数の不足などから管理が難しくなっていた部分がありました。

そこに興味を持った私が、技術的な部分を専門的に見ていくために入部したことで技術班が設立されました。

特に外から見える成果としては、VLL オフィシャルサイト および ライブごとの特設 Web サイトを、
見た目を変えることなくコード / インフラともにゼロからやり直すことで、大幅な高速化とゼロコストを実現しました。

<UrlCard url="https://mikuec.com" />

同時に GitHub の運用や CI / CD をはじめとした自動化の導入、用途に適したモダンな技術の導入も進めることで、
来年度以降も持続的に開発を続けるための地盤整備を行いました。

しかし、ライブ特設サイトの制作だけでなく部内 Wiki の Notion 移行やその他情シス的なタスクなど
多くの仕事を捌いていくなかで、特にプログラミングが必要な部分において私しか触れない部分が増えていっていました。
